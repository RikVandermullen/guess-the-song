# Guess The Song

Project where you need to guess songs based on a snippet. Made with:
- Angular
- NestJS
- Neo4j

## Functionalities:
- Create a profile and login
- CRUD (artists, songs and games)
- Highscores and leaderboards
- Song recommendations
