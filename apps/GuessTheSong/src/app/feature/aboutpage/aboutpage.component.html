Guess The Song

This website is a place where you can test your knowledge for songs. We all know that one friend that thinks he knows the title of every song. Well, this is the place to prove it! As a logged in User you can play games to test your knowledge. You can also create your own games for other players or friends to play. An User has a Name, Email Address, Password, Phone Number and a Date of Birth.


A game contains a chosen amount of songs which play for 30 seconds. You need to make your guess in this time frame otherwise you won't score any points! At the end of the game you will see your score and the correct answers. You can also see the scores of other players on a leaderboard. On your profile page you can find and edit your information and see some statistics about your games like your total score, averge winpercentage and more!


When creating a game, you are given three options: Select your own song, Random and Recommended. With the option select your own, you can choose the songs yourself based on a chosen Genre. With the random option, you can choose a random amount of songs also based on a Genre. With the option recommended, you can choose a random amount of songs that are recommended for you. This is based on your previous games and the songs you have played. The more you play, the more songs will be recommended for you! Other fields that need to be filled in are: a name, a description and the genres. You can also choose to make the game public or private. If you make the game public, everyone can play it. If you make it private, only you and your friends can play it.


You can also add songs which aren't yet on the website. You need to fill in the Title, Artist, Album, Publish date, Genres, Cover image and the link for the song preview. These links can be found on Apple Music. If an Artist is missing, you can also add them. For this you need to fill in the Name, a Description and an Image of the artist.


Scores will be generated automatically by the system. A Score contains the amount of right answers, the amount of time it took for the game to complete , the date the game was played and the final score. The final score is calculated by the amount of right answers and the amount of time it took to complete the game.

Entity Relationship Diagram

Some information will be stored in a NoSQL database. The entities are shown below. A Score is a weak entity which is dependent on a Game and an User. An User scores an Score and a Game can have many Scores. A Song can be added to multiple Games and a Game can have multiple Songs. An Artist can have multiple Songs but a Song can only have one Artist.

User Stories

Nr Story Extra
1 As an User which is not logged in, I can see the games, artists and songs but not do any other actions with them.
2 I can create an account for this website by registrating and filling in all the necessary information. A valid Email and Date of Birth must be given.
3 As an User which is logged in, I can play games and get scores. After the game is finished a leaderboard will be displayed.
4 As an User which is logged in, I can view my profile page where I can see and edit my information. I can also find statistics about my games and I have to ability to delete my account.
5 As an User which is logged in as an Admin, I can add songs by filling in all the necessary information. I can view in real time how to song will be displayed on the website. Users which are Players can only see Song details
6 As an User which is logged in as an Admin, I can add artists by filling in all the necessary information. I can view in real time how to artist will be displayed on the website. Users which are Players can only see Artist details
7 As an User which is logged in as a Player or Admin, I can create games by filling in all the necessary information. I can also choose to make the game private or public. I can only edit or delete games which I added and not have been played. Public games can be played by other players, private can't.
8 A Game can be created with 3 different game modes: Select your own songs, Random songs and Recommended songs Random songs will be added based on the chosen Genre. Recommended songs will added based on other players which have guessed the same songs
9 Scores will be automatically generated by a formula based on correct answers and leftover time.
10 A leaderboard is displayed on the home page with users which have the highest scores. The leaderboard will update once a new user needs to be displayed.
11 A leaderboard is displayed after the game ends with users which have the highest scores. The leaderboard will update once a new user needs to be displayed.
